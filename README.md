# Simple-JavaScript-Calculator

## Description
A browser-based calculator built with HTML and JavaScript. The program loop prompts the user for two numbers and an arithmetic operator, computes the result, and displays the inputs and results in a styled table when exiting the loop (cancel). After the user exits, a summary table displays the minimum, maximum, average, and total of all valid results.

## Files
- `calculator.html` - loads and executes the JavaScript file
- `calculator.js` - contains all logic for input handling, computation, table construction, and styling

## How to Use
1. Open `calculator.html` in a browser
2. Enter the first number when prompted
3. Enter an operator (+, -, %, /, *)
4. Enter the second number when prompted
5. Click **OK** to continue adding calculations or **Cancel** to stop and view the tables

## Features
- Supports addition, subtraction, modulus, division, and multiplication
- Displays an error message for invalid operators
- Displays an error message for non-numeric inputs
- Summary table shows stats for valid results only
- Orange styled tables matching assignment design

## Technologies Used
- HTML
- JavaScript
- CSS (inline via JavaScript)
